# Personal Portfolio Website

This is a static HTML and CSS portfolio site designed to be hosted with GitHub
Pages.

## Pages

- `index.html` - work experience and education
- `projects.html` - project showcase
- `contact.html` - contact information and links
- `styles.css` - shared design and layout

## Customize

Replace the placeholder text in the HTML files with your real:

- name
- work experience
- college information
- high school information
- contact details
- project links

## Publish With GitHub Pages

1. Push these files to a GitHub repository.
2. Open the repository on GitHub.
3. Go to `Settings` > `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and the `/ (root)` folder.
6. Save and wait for GitHub to publish the site.

Your website will then be available at a GitHub Pages URL.
